# 课程介绍

这里用于记录当前课程的学习过程、核心概念、实践步骤和阶段性总结。

## 学习目标

- 理解课程涉及的业务背景、需求推导和架构设计方法
- 梳理服务端内核引擎、工程化建设和前端领域模型的实现过程
- 沉淀可复用的组件库、npm 包封装和项目实践经验

## 学习路径

- 1.  前言
- 2.  行业分析与需求推导
- 3.  技术选型与架构设计
- 4.  基于 nodejs 实现服务端内核引擎
- 5.  基于 webpack5 完成工程化建设
- 6.  基于 vue3 完成领域模型架构构建
- 7.  基于 vue3 完成动态组件库建设
- 8.  完成框架 npm 包抽象封装并发布
- 9.  【赠课扩展】框架应用与项目实践
